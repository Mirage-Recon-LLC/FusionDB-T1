Getting Started

Installation Guide

🪟 WINDOWS SETUP

1
Run the Installer
Double-click fusiondb-installer.exe. The installer bootstraps the runtime and adds fusiondb.exe to your PATH.
2
Generate Your Encryption Key
FusionDB requires a 32-byte master key. Generate one and set it as an environment variable.
openssl rand -hex 32 $env:FUSIONDB_SECRET = "your-64-hex-chars"
3
Fuse Your First Entity
Create a UFL manifest JSON file and fuse it.
fusiondb -db=./mydata ufl manifest.json

🐧 LINUX SETUP

1
Make Executable and Run
Mark the bootstrap binary executable and run it to install FusionDB.
chmod +x fusiondb-bootstrap-linux ./fusiondb-bootstrap-linux
2
Set Your Encryption Key
Generate a secure 32-byte key and export it. Store this in a secrets manager — data cannot be decrypted without it.
export FUSIONDB_SECRET=$(openssl rand -hex 32)
3
Start the Observability Server
Launch the built-in HTTP server for health and readiness probes.
fusiondb -db=./data serve --port 8080
